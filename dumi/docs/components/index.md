# Hiller-Design

`Hiller-Design` 是一个轻便的 `React UI` 组件库，主要用于研发企业级中后台产品。

## Component Demo

```tsx
import React from 'react';
import { Alert } from '../../../src/components';

const App = () => {
  return (
    <Alert>Hiller Design</Alert>
  );
}

export default App;
```
